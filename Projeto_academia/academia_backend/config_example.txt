# INSTRUÇÕES PARA CONFIGURAÇÃO DO PROJETO

## 1. Criar arquivo .env na raiz do projeto (academia_backend/.env)

Copie o conteúdo abaixo para um arquivo chamado .env:

```
# Configurações do Django
SECRET_KEY=django-insecure-sua-chave-secreta-aqui-mude-em-producao-123456789
DEBUG=True

# Configurações do Banco de Dados (opcional - usando SQLite por padrão)
# DATABASE_URL=sqlite:///db.sqlite3

# Configurações de Email (opcional)
# EMAIL_HOST=smtp.gmail.com
# EMAIL_PORT=587
# EMAIL_USE_TLS=True
# EMAIL_HOST_USER=seu-email@gmail.com
# EMAIL_HOST_PASSWORD=sua-senha-de-app

# Configurações de CORS (opcional)
# CORS_ALLOWED_ORIGINS=http://localhost:3000,http://127.0.0.1:3000

# Configurações de JWT (opcional)
# JWT_ACCESS_TOKEN_LIFETIME=60
# JWT_REFRESH_TOKEN_LIFETIME=7
```

## 2. IMPORTANTE: 
- NUNCA commite o arquivo .env para o Git
- Mude a SECRET_KEY em produção
- Configure DEBUG=False em produção
